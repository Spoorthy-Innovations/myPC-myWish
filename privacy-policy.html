Privacy Policy

myPC myRight & Pro

Privacy policy for the myPC myRight and myPC myRight Pro browser extensions. This page explains what data is collected (none), how the extensions work, and how your privacy is protected.

Overview

myPC myRight and myPC myRight Pro are browser extensions that allow users to force copy, paste, text selection, right‑click, and temporary password visibility on webpages that try to block these standard browser features. Pro adds advanced controls such as visibility bypass, keyboard unblock (e.g. DevTools, print, view source), overlay removal, drag & drop, print unlock, scroll unlock, video controls, autocomplete enforcer, exit-dialog bypass, and element zapper. Both extensions work entirely locally; neither collects nor transmits any data.

Data collection

myPC myRight and myPC myRight Pro do not collect, store, or transmit any personal or usage data.

  • No browsing history is collected.
  • No page contents, form data, or passwords are collected.
  • No analytics, tracking, or advertising cookies are used.

All logic runs locally in your browser. The extension never sends information to external servers.

How the extensions work

myPC myRight and myPC myRight Pro inject a small content script on webpages in order to restore standard browser behaviors that some sites try to block, such as copy, paste, selection, right‑click, and optional password visibility. Pro adds a second script for advanced features (e.g. overlay removal, keyboard unblock).

The scripts:

  • Run locally in the browser.
  • Adjust event handlers and CSS rules that prevent normal actions.
  • Do not read or transmit the contents of the page.
  • Do not inject or execute any remote code.

Permissions

Both extensions use a minimal set of permissions:

  • Host permissions (<all_urls>) – required so the extension can work on any site that blocks copy, paste, selection, right‑click, or password visibility. The extension does not read or export page content; it only modifies how blocking scripts behave.
  • activeTab – used so the extension can apply its behavior only to the tab the user chooses to interact with.
  • storage – used only to save the user’s own settings (e.g. which features are enabled; on Pro, Advanced and Power Tools toggles). These settings remain local to the browser.

Contact

If you have questions about this privacy policy or how myPC myRight or myPC myRight Pro handle data, you can contact the developer using the details provided on the Chrome Web Store listing or project website.

No data collection, tracking, or sharing.
